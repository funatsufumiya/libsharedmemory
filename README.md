# libsharedmemory (lsm)

`libsharedmemory` is a small C++11 header-only library for using shared memory on Windows, Linux and macOS. It allows for simple read/write data transfer of `uint8_t*` / `unsigned char*` and `std::string`.

# Build

```sh
./build.sh
```

# Test
